# 三国杀91助手

为网页版《三国杀一将成名》玩家制作的油猴助手。

目前支持谋邓艾、凌烈、张裕、胡班、桓阶的技能提示和手牌花色排序。脚本只读取当前游戏页面中的本地数据，不会自动点击、自动出牌或上传对局数据，也不依赖任何其他油猴脚本。

## 一键安装

安装前请确保浏览器已经安装 [Tampermonkey](https://www.tampermonkey.net/) 或 Violentmonkey。

**[点击安装三国杀91助手](https://github.com/doctrine5431/sgs_91tools/releases/latest/download/sgs91-assistant.user.js)**

打开链接后，在油猴扩展的安装页面点击“安装”，然后刷新游戏网页即可。

也可以进入 [Releases](https://github.com/doctrine5431/sgs_91tools/releases) 查看版本说明和下载历史版本。

## 已支持功能

### 武将提示

- 谋邓艾「骤袭」：提示骤袭 X 数量、可用和失效状态。
- 凌烈「狩虎」：提示技能当前状态。
- 张裕「相谶」：提示技能选择的目标。
- 胡班「崇义」：提示首张牌优先使用【杀】。
- 桓阶「谏立」：提示剩余发动次数和失效状态。

### 通用功能

- 手牌花色排序：开启“91”悬浮按钮后，单击即可整理自己的手牌显示顺序。
- 悬浮按钮开关：“91”按钮默认关闭，可在油猴扩展的脚本菜单中开启或关闭，并记住选择。

## 支持的游戏页面

```text
https://web.sanguosha.com/*
https://*.sanguosha.com/*
```

## 使用方法

1. 安装脚本并打开网页版三国杀。
2. “91”悬浮按钮默认关闭；需要时在油猴扩展的“三国杀91助手”菜单中选择“开启 91 悬浮窗”。
3. 使用支持的武将时，对应技能提示会自动显示。
4. 需要整理手牌时，单击“91”按钮；按住按钮可以拖动位置。

## 常见问题

### 没有出现“91”按钮

“91”悬浮按钮默认关闭。请打开油猴扩展菜单，展开“三国杀91助手”，选择“开启 91 悬浮窗”。开启状态会被记住；仍未出现时请等待游戏场景加载，或刷新游戏页面。

### 武将提示没有出现

请确认当前使用的是受支持武将，并已进入提示所需的游戏阶段。如果谋邓艾提示仍无法识别，可按 `F12` 打开控制台并运行：

```javascript
MouDengAiJuxiHelper.copyDiagnostic()
```

诊断内容只会复制到剪贴板，由玩家自行决定是否提供。可以前往 [Issues](https://github.com/doctrine5431/sgs_91tools/issues) 反馈问题。
