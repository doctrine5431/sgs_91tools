# 三国杀91助手

为网页版《三国杀一将成名》玩家制作的油猴助手。

目前支持谋邓艾「骤袭」提示和手牌花色排序。脚本只读取当前游戏页面中的本地数据，不会自动点击、自动出牌或上传对局数据。

## 一键安装

安装前请确保浏览器已经安装 [Tampermonkey](https://www.tampermonkey.net/) 或 Violentmonkey。

**[点击安装三国杀91助手](https://raw.githubusercontent.com/doctrine5431/sgs_91tools/main/dist/%E4%B8%89%E5%9B%BD%E6%9D%8091%E5%8A%A9%E6%89%8B.user.js)**

打开链接后，在油猴扩展的安装页面点击“安装”，然后刷新游戏网页即可。

## 支持的游戏页面

```text
https://web.sanguosha.com/*
https://*.sanguosha.com/*
```

## 当前功能

### 谋邓艾「骤袭」提示

- 自动识别自己是否为谋邓艾或拥有「骤袭」。
- 在自己的出牌阶段计算「骤袭」需要的 X 数量。
- 在对应手牌上标记可以计入 X 的牌。
- 在自己的武将位置显示当前「骤袭 X=数量」。
- 自己或其他谋邓艾发动骤袭但弃牌少于三张时，显示“骤袭已失效”，并在其回合结束后清除。
- 不会替玩家发动技能、选择目标或打出卡牌。

### 手牌花色排序

- 游戏页面右下方会显示一个可拖动的“91”按钮。
- 单击按钮即可按游戏内部花色编号整理自己的手牌显示顺序。
- 只改变本地画面中的排列，不会使用、丢弃或移动真实卡牌。

## 使用方法

1. 安装脚本并打开网页版三国杀。
2. 进入对局后等待“91”按钮出现。
3. 使用谋邓艾时，骤袭数值和手牌标记会自动显示。
4. 需要整理手牌时，单击“91”按钮；按住按钮可以拖动位置。

## 安全说明

- 不自动点击，不自动出牌。
- 不上传账号信息或对局数据。
- 不请求外部接口，也不加载远程代码。
- 所有计算和显示都在当前浏览器页面内完成。

本项目仅用于提供本地信息提示和界面整理，请遵守游戏平台规则。

## 常见问题

### 没有出现“91”按钮

游戏场景可能还没有加载完成，请稍等几秒。如果仍未出现，可以刷新游戏页面。

### 谋邓艾提示没有出现

请确认当前使用的是谋邓艾，并已进入自己的出牌阶段。如果仍无法识别，可按 `F12` 打开控制台并运行：

```javascript
MouDengAiJuxiHelper.copyDiagnostic()
```

诊断内容只会复制到剪贴板，由玩家自行决定是否提供。可以前往 [Issues](https://github.com/doctrine5431/sgs_91tools/issues) 反馈问题。
